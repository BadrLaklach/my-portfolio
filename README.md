## 🚀 À propos

Élève Ingénieur en 4ème année (Filière Génie Logiciel et Systèmes Intelligents) à l'ENSAM Meknès. Ce portfolio présente mes projets (Data Science, Intelligence Artificielle, et Développement Mobile/Web), mes compétences DevOps/Cloud, ainsi que mes expériences professionnelles et certifications.

## 🛠️ Stack technique du Portfolio

- **Framework** : Next.js 16 (App Router)
- **Langage** : TypeScript
- **Style** : Tailwind CSS 4
- **Animations** : Three.js (React Three Fiber)
- **Composants UI** : Radix UI (shadcn/ui)
- **i18n** : Détection automatique et bascule français/anglais
- **Déploiement** : GitHub Pages

## ✨ Fonctionnalités

- **Animations 3D** : Effets de fond interactifs avec Three.js
- **Interface bilingue** : Support complet FR/EN via détection du navigateur
- **Design responsive** : Parfaitement adapté mobile, tablette et desktop
- **Mode sombre/clair** : Thème adaptatif avec persistance
- **Contact Rapide** : Call-to-action statique générant des emails pré-formatés
- **SEO optimisé** : Métadonnées, robots.txt, et JSON-LD
- **Pages légales** : Mentions légales et politique de confidentialité

## 🌍 Internationalisation

Le site détecte automatiquement la langue du navigateur :
- 🇫🇷 Français si le navigateur est en français
- 🇬🇧 Anglais dans tous les autres cas (fallback par défaut)

Les traductions sont gérées via `lib/translations.ts` et le hook `useLanguage()`.
